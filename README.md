# AI-KaiHei
和AI队友开黑，打游戏！

## 简介

项目中文名称：AI开黑

项目英文名称：AI Kai Hei

项目方向：人工智能 开黑 游戏队友

短期目标：用 人工智能 玩荒野乱斗游戏 和 自己开黑

运行环境：项目运行在windows10上，android模拟器 + windows 10
